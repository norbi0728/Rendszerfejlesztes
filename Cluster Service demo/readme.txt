1. A clusterService.exe elindítása
2. Meg kell várni, hogy megjelenjen az output a konzol ablakban
 (azt fogja tartalmazni, hogy elindult a szerver és hogy hol lehet elérni,
  ez első indításnál lehet 1 perc is akár, ezek után ezzel már nincs teendő
  max a végén bezárni ezt a konzolablakot)
3. Ha megjelent a konzolon a program outputja, lehet indítani a java
  programot, ami egy IntelliJ maven project. 
4. Indítás után a billentyűzetről fogja kérni az adatokat, egyesével, 
  az output jelezni fogja, hogy melyik attribútumot várja.
5. Az utolsó attribútum megadása után, a program elküldi a szervernek a 
  megadott attribútumokat és megjeleníti a választ
